# 🏡 État des Lieux

Application mobile pour réaliser des états des lieux.

## Déploiement sur Vercel

### Méthode simple (sans code) :

1. Crée un compte sur [github.com](https://github.com) (gratuit)
2. Crée un nouveau repository "etat-des-lieux"
3. Upload tous ces fichiers dans le repository
4. Va sur [vercel.com](https://vercel.com) → "New Project"
5. Connecte ton GitHub → sélectionne "etat-des-lieux"
6. Clique "Deploy" → ton app est en ligne !
